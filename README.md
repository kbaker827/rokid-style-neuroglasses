# Rokid Style NeuroGlasses

Audio-first GPT-4o assistant for Rokid AI Glasses Style.

This Android/Kotlin port adapts NeuroGlasses from an iOS companion plus HUD bridge into a standalone Style app. It uses push-to-talk recording, OpenAI transcription, Chat Completions, and OpenAI TTS playback.

## Features

- Push-to-talk audio capture
- OpenAI Whisper transcription
- Chat Completions with prompt presets
- OpenAI TTS playback queue
- Minimal settings for API key, model, base URL, voice, and streaming

## Build

Open the folder in Android Studio and build the `app` module.

## Notes

This is designed for display-free use: hold to talk, release to process, hear the answer.
